Installation

1. Extract packages.zip and GUTGenerator.zip
2. Copy all content of folder packages into GUTGenerator/packages folder
3. Open project, build and run

How to run

1. Put commandline arguments into Debug section of project GUTGenerator.App
   /path/to/a/folder/of/MCALConfGen.exe/and/module.dll /path/to/testspec.xlsx /path/to/output/folder [sheet_name]
2. [sheet_name] is optional, if it's used, only 1 sheet is generated. Otherwise, all of sheets are generated
